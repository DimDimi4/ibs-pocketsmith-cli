# Usage

<!-- usage -->
```sh-session
$ npm install -g ibs-pocketsmith
$ ibs-ps COMMAND
running command...
$ ibs-ps (-v|--version|version)
ibs-pocketsmith/0.0.0 win32-x64 node-v14.16.0
$ ibs-ps --help [COMMAND]
USAGE
  $ ibs-ps COMMAND
...
```
